int PHActsTrkProp::Process ( )
virtual

Process each event by calling the fitter.

Collect all source links for the CKF

If we are resetting the covariance and the space point to the vertex, the POCA should have the covariance of the vertex

Construct a perigee surface as the target surface This surface is what Acts fits with respect to, so we set it to the initial vertex estimation

Construct the options to pass to the CKF. SourceLinkSelector set in Init()

Run the CKF for all source links and the constructed track seed CKF runs both track finder and KF track fitter

void PHActsTrkProp::setupSourceLinkSelection ( )
private

Setup the source link selector criteria.

The CKF requires a source link selector which helps it identify possible SLs to the track. The selections can be added like {makeId(volId, layerId), {maxChi2, numSourceLinks}} We'll just put the max chi2 to 10

Global detector criteria

Volume criteria Volume IDs - MVTX = 7, INTT = 9, TPC = 11

Set individual layer criteria (e.g. for first layer of TPC) Individual layers should only have one SL per layer

Vol IDs are 7, 9, 11, hence vol*2+7

void PHActsTrkProp::updateSvtxTrack ( Trajectory  traj,
const unsigned int  trackKey,
Acts::Vector3D  vertex 
)
private

Wipe and recreate the SvtxTrackMap with Acts output.

There could be multiple tracks per trajectory found from the CKF Therefore we need to get the info from the original SvtxTrack that we need, and then create new tracks

All these tracks should have the same vertexId if they came from the same track seed

hack for now to just set the vertex id to 0 if it is needed

Create a state at 0.0 to propagate out from

This gets the track indexer and associated tracks (Acts::MultiTrajectories)

Iterate over the found tracks via their trackTip index

No trackParameters for this trackTip, so fit failed for this tip

If it is the first track, just update the original track seed

Get the last track key so that we can add new tracks if needed

Otherwise make a new track

Needs to be a new track id

If negative QOP then set charge to negative

Now insert the trajectory + trackTip-trackKey pair
