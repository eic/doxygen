Plot the material on each surface as function of Phi and Eta (two 1D plots). If a surface map json file is specify it is parse to associate name to the different surface id. nbprocess : number of parameter to be processed. name : name of the output directory. The parsing of the Json surface map file (use to associate the name to the volumes) might not work with version of root newer that version 6.18.04
