Detailed Description

This class is responsible for taking SvtxTracks and converting them to track seeds that Acts can take in to the track fitter. It collects SvtxTracks, converts them to Acts tracks, and then finds the corresponding TrkrClusterSourceLinks to that SvtxTrack. The output is a node on the node tree that is a map of Acts track seeds and corresponding source links
