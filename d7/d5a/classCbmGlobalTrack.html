Detailed Description

CbmGlobalTrack.h

Author
V.Friese v.fri.nosp@m.ese@.nosp@m.gsi.d.nosp@m.e

Data class for Global CBM track. Data level RECO. It consists of local tracks in STS, MUCH and TRD and a RICH ring.
